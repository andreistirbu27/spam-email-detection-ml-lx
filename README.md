# Spam Email Detection ML

A machine learning project that detects whether an email or message is spam/scam or legitimate using NLP preprocessing, TF-IDF feature extraction, and supervised classification models.

## Project Structure

spam-email-detection-ml-lx/
├── data/
│   ├── raw/
│   └── processed/
├── models/
├── notebooks/
├── reports/
│   └── figures/
├── src/
├── tests/
├── README.md
├── requirements.txt
└── .gitignore

## Goal

Build and evaluate machine learning models for spam email/message classification.

## Planned Models

- Naive Bayes
- Logistic Regression
- Support Vector Machine
- Random Forest

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
